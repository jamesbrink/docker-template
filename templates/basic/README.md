## About

This is basic boilerplate for a docker based project. 

## Building

To build the project:
```shell
make
```

To list the images:
```shell
make list
```

To run any tests:
```shell
make test
```

To push image to remote docker repository:
```shell
docker login ....
make push
```

To cleanup and remove built images:
```shell
make clean
```

## Usage

To run the container:
```shell
docker run -i -t utensils/template
```
